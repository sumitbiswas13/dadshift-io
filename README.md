# DadShift.io

> Every shift counts. Both of them.

A community blog platform for dads navigating careers, parenthood, and the beautiful chaos in between.

## Features
- 📝 Write & publish blog posts with rich text editor
- 💬 Comment on posts
- ❤️ Like posts
- 👤 Dad profiles
- 🏷️ Categories: Balance, Dad Wins, Tough Days, Work & Life, Milestones, Advice
- 🔐 Google OAuth sign-in
- 🔥 Firebase Firestore backend

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS
- Firebase (Auth + Firestore)
- TipTap rich text editor
- React Router v6

## Setup
1. Clone the repo
2. `npm install`
3. Add your Firebase config to `src/lib/firebase.ts`
4. `npm run dev`
